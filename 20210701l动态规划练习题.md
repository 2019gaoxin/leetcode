```java
小朋友 A 在和 ta 的小伙伴们玩传信息游戏，游戏规则如下：

有 n 名玩家，所有玩家编号分别为 0 ～ n-1，其中小朋友 A 的编号为 0
```

### 动态规划dp数组题解

首先这个问题是计数，这类题明显有两个变量一个n,一个k此时应该想到用二维dp数组解题，现在思考最基本的dp数组的状态转移方程的设计。

一般都是从0出发的，第0轮

那么dp 【0】【】就为第0轮，他的其实地址就是上一轮的最终地址所以它存的是一个最终地址的方案数。

dp【k】【n-1】即为最终的结果。





```java
//利用dp数组解决问题，
/*
	边界条件：能不能到达0；
	 relation = [[0,2],[2,1],[3,4],[2,3],[1,4],[2,0],[0,4]], k = 3
	 解释：信息从小 A 编号 0 处开始，经 3 轮传递，到达编号 4。共有 3 种方案，分别是 0->2->0->4， 0->2->1->4， 0->2->3->4。
	最后的一个dp储存的结果为最终结果
	
	0->2->0->4
	0->2->1->4
	0->2->3->4       结果是3；
	dp[轮数][到哪个小朋友]
	dp[][] = 0 或1；
	
动态规划的状态转移方程：
	dp[i+1][to] =+ dp[i][from]；
	
1.	
dp[0][0] = 1,第0轮到0号小朋友的总方案数
2.
dp[1][]  这是第一轮的
r[0] 和 r[1] 分别为[0,2] 中的0和2。
dp[1][2] = dp[1][2] + dp[0][0] =0 +1;第一轮到2号小朋友的方案数。
dp[1][1] = dp[1][1] + dp[0][2] =1 +0;第一轮到1号小朋友的方案数。
dp[1][4] = dp[1][4] + dp[0][3] =0;第一轮到4号小朋友的方案数。
dp[1][3] = dp[1][3] + dp[0][2] =0;第一轮到3号小朋友的方案数。
dp[1][4] = dp[1][4] + dp[0][1] =0;第一轮到号小朋友的方案数。
dp[1][0] = dp[1][0] + dp[0][0] =1;第一轮到号小朋友的方案数。

dp[2][]:第二轮。
dp[2][2] = dp[2][2] + dp[1][0] =0 +1;第一轮到2号小朋友的方案数。

dp[k][n-1]    : k为多少轮，n-1为目的数

dp[][]  如果是通路那么就是1 如果不是就置为0.如果一直为1，这样就可以达到累加的效果
如果是通路且只有1条则1+0+0+0



*/
class Solution {
    public int numWays(int n, int[][] relation, int k) {

        int[][] dp = new int[k+1][n];
        dp[0][0] = 1;
        for(int i = 1;i<=k;i++){
            for(int[] r:relation ){
                dp[i][r[1]]+=dp[i-1][r[0]];
            }
        }

        return dp[k][n-1];

    }
}
```
